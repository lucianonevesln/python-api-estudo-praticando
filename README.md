# Estudo 2 : Python API Cadastro 2 (praticando)

## Objetivo :

### Reforçar e ampliar os estudos iniciados em "https://github.com/lucianonevesln/python-api-estudo", de como criar uma API com operação CRUD (Create, Read, Update e Delete) conectada a um banco de dados MySQL.

## Tecnologias Usadas :

### - Python;

### - Flask;

### - SQLAlchemy;

### - MySQL;

### - Postman;

## Comandos Úteis :

### - python -m venv venv : comando para criar um ambiente virtual para a aplicacão;

### - .\venv\Scripts\activate : comando para executar o ambiente virtual;

### - deactivate : comando parar desativar o ambiente virtual;

### - pip install flask : comando para baixar o microframework Flask;

### - pip install SQLAlchemy: comando para baixar a biblioteca de conexão com o banco de dados;

### - pip install flask_sqlalchemy: comando para baixar esse modulo;

### - mysql-connector-python : comando para baixar esse modulo;

### - mysqlclient : comando para baixar esse modulo;

### - python servidor.py : comando para executar a aplicação;

### - Ctrl + C : comando para desativar a aplicação;

### - from servidor import db : comando para ser executado no cmd, que será utilizado para criar a tabela em banco de dados;

### - db.create_all() : comando que cria a tabela em banco de dados;

### - exit() : comando para sair do ambiente python;
